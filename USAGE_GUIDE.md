# Anime Video Generator - 使用指南

## 快速开始

### 1. 获取 API 密钥
1. 访问 [fal.ai](https://fal.ai) 注册账户
2. 获取您的 API 密钥
3. 确保账户有足够的余额用于视频生成

### 2. 配置应用
1. 启动应用后，点击右上角的"Settings"按钮
2. 在"Fal.ai API Key"字段中输入您的 API 密钥
3. 点击"Validate Key"验证密钥是否有效
4. 设置默认的分辨率、时长和宽高比
5. 点击"Save Settings"保存设置

### 3. 生成您的第一个视频

#### 文本到视频
1. 确保选择了"Text to Video"模式
2. 在提示框中输入详细的描述，例如：
   - "一个蓝发的动漫角色在魔法森林中奔跑"
   - "动漫风格的少女在霓虹灯闪烁的城市夜景中跳舞"
3. 选择分辨率、时长和宽高比
4. 点击"Generate Video"
5. 等待30-60秒完成生成

#### 图片到视频
1. 选择"Image to Video"模式
2. 上传一张图片（支持JPG、PNG、WEBP、GIF、AVIF格式）
3. 输入描述如何让图片动起来的提示
4. 选择分辨率和时长
5. 点击"Generate Video"

## 功能详解

### 视频生成设置

#### 分辨率选项
- **480p (Lite)**: 低成本选项，约$0.18/5秒视频
- **720p (Lite)**: 中等质量，约$0.18/5秒视频  
- **1080p (Pro)**: 高质量，约$0.74/5秒视频

#### 时长选项
- **5秒**: 标准时长，成本较低
- **10秒**: 较长时长，成本翻倍

#### 宽高比选项
- **16:9**: 横屏格式，适合桌面观看
- **9:16**: 竖屏格式，适合手机观看
- **1:1**: 正方形格式，适合社交媒体

### 提示词技巧

#### 好的提示词示例
- "一个穿着校服的动漫少女在樱花飞舞的校园里走路"
- "机甲战士在未来城市的天空中飞行，背景是霓虹灯"
- "可爱的动漫猫咪坐在窗台上看雨，温馨的室内场景"

#### 提示词建议
- 使用具体的描述而不是抽象概念
- 包含动作词汇（跑步、跳舞、飞行等）
- 描述场景和环境
- 指定动漫风格特征
- 保持在500字符以内

### 视频管理

#### 查看视频
- 生成完成后，视频会显示在右侧播放器中
- 点击"Gallery"标签查看所有生成的视频
- 使用筛选和排序功能管理视频

#### 下载视频
- 点击视频播放器下方的"Download"按钮
- 视频将以MP4格式保存到您的下载文件夹

#### 删除视频
- 在画廊中点击视频下方的"Delete"按钮
- 确认删除后视频将从本地存储中移除

## 成本管理

### 价格参考
- **1080p Pro**: 每5秒视频约$0.74
- **720p/480p Lite**: 每5秒视频约$0.18
- **10秒视频**: 成本是5秒视频的2倍

### 节省成本的建议
1. 从低分辨率开始测试提示词
2. 确认效果满意后再使用高分辨率
3. 使用5秒时长进行初步测试
4. 定期检查Fal.ai账户余额

## 常见问题

### Q: 视频生成失败怎么办？
A: 检查以下几点：
- API密钥是否正确配置
- 账户余额是否充足
- 网络连接是否稳定
- 提示词是否过于简单或复杂

### Q: 生成速度很慢？
A: 视频生成通常需要30-60秒，这是正常的。高分辨率和长时长会增加生成时间。

### Q: 如何提高视频质量？
A: 
- 使用更详细和具体的提示词
- 选择1080p分辨率
- 确保上传的图片质量良好（用于图片到视频）

### Q: 支持哪些图片格式？
A: 支持JPG、JPEG、PNG、WEBP、GIF、AVIF格式，文件大小限制为10MB。

### Q: 视频可以商用吗？
A: 请查看Fal.ai和ByteDance的服务条款，了解具体的使用许可。
